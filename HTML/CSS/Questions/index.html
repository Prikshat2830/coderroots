Q.6: Diff. b/w let,const and var

ANS: LET- let variable is a block scope. It can be updated but can't be redeclared. In let hoisting can not be initialized.
CONST- const variable is also a block scope. But it cannot be redeclared and update.
VAR- var variable is a funtion scope. It can be redeclared and also be updated. Hoisting is initialized in var.

Q.7: Hoisting and where is possible?

ANS: Hoisting is that when javascript moves the declaration to the top of the scope before execution. Hoisting is done in all the variables (let,const,var). In var is hoisted and is initialized with undefined, whereas the let and const variable are also hoisted but cannot be initialized.
